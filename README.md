## Summary
A small **React + TypeScript** app to browse popular GitHub repositories.  
It fetches one page at a time (sorted by stars), supports simple pagination, local starring (saved in `localStorage`), and an optional language filter.

---

## Features
- Fetch popular repos from **GitHub Search API** (one page per request)  
- Paginate results  
- Star/unstar repos locally (**persisted in localStorage**)  
- Filter by language (e.g., `language:TypeScript`)  
- Follow **BEM CSS** and tested logic
